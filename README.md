# KS IT Solution - ব্রডব্যান্ড ইন্টারনেট ওয়েবসাইট

## 🚀 কীভাবে ব্যবহার করবেন

### 1. ওয়েবসাইট চালু করা
- যেকোনো HTML ফাইল ব্রাউজারে ওপেন করুন (index.html থেকে শুরু করুন)
- অথবা Live Server ব্যবহার করুন (VS Code Extension)

### 2. তথ্য পরিবর্তন করা
প্রতিটি HTML ফাইলে নিচের তথ্য পরিবর্তন করুন:

**ফোন নম্বর:** `০১৭০০-০০০০০০` খুঁজে আপনার নম্বর দিন
**ইমেইল:** `info@ksitsolution.com` পরিবর্তন করুন
**ঠিকানা:** "গাইবান্ধা সদর, গাইবান্ধা" পরিবর্তন করুন
**WhatsApp নম্বর:** `8801788213414` পরিবর্তন করুন (দেশ কোড সহ)

### 3. Google Maps এডিট করা
[contact.html](contact.html#L200) ফাইলে Google Maps iframe এর `src` পরিবর্তন করুন:
1. Google Maps এ যান
2. আপনারঅফিসের অবস্থান খুঁজুন
3. Share > Embed a map > Copy HTML
4. iframe এর src কপি করে contact.html এ পেস্ট করুন

### 4. প্যাকেজের দাম পরিবর্তন
[packages.html](packages.html) এ প্রতিটি প্যাকেজের দাম, স্পিড ও বিবরণ পরিবর্তন করুন।

### 5. ছবি যুক্ত করা
`images/` ফোল্ডারে আপনার ছবি রাখুন এবং HTML ফাইলে পাথ আপডেট করুন।
